# Pong 21

Консольная реализация классической игры «Понг» на языке C. Двое игроков управляют ракетками с двух сторон поля и отбивают мяч. Побеждает тот, кто первым наберёт **21 очко**.

## Особенности

- Игровое поле в терминале размером 80×25 символов, отрисованное ASCII-графикой.
- Игра для двух игроков за одной клавиатурой.
- Пошаговый игровой цикл: игроки вводят команду, после чего поле обновляется.
- Автоматический отскок мяча от верхней и нижней стенок.
- Подсчёт очков и определение победителя.

## Сборка

Для компиляции нужен любой компилятор C (например, `gcc` или `clang`):

```sh
gcc -o pong21 pong21.c
```

или

```sh
clang -o pong21 pong21.c
```

## Запуск

```sh
./pong21
```

## Управление

Игра пошаговая. За один ход можно ввести одну команду (одну букву), затем нажать **Enter**.

| Игрок    | Вверх | Вниз |
|----------|:-----:|:----:|
| Игрок 1 (левая ракетка)  | `A` | `Z` |
| Игрок 2 (правая ракетка) | `K` | `M` |

- Регистр не важен: `A` и `a` работают одинаково.
- Нажатие **Enter** без буквы — пропуск хода (мяч всё равно двигается).
- При вводе недопустимого символа выводится сообщение `Invalid input`, и ход нужно повторить.

## Правила

- Мяч (`0`) движется по полю и отскакивает от верхней и нижней границ.
- Если игрок отбивает мяч своей ракеткой (`|`), мяч летит в обратную сторону.
- Если мяч пролетает мимо ракетки и уходит за границу, соперник получает **1 очко**, а мяч возвращается в центр поля.
- Игра заканчивается, когда один из игроков набирает **21 очко**. Победитель объявляется в финальном сообщении.

## Обозначения на поле

| Символ | Значение                     |
|:------:|------------------------------|
| `-`    | верхняя и нижняя границы поля |
| `\|`   | боковые стенки, центральная линия и ракетки |
| `0`    | мяч                          |

## Технические детали

- Размер поля: `80 × 25` (константы `WIDTH`, `HEIGHT`).
- Длина ракетки: `3` символа (`RACKET_SIZE`).
- Очки для победы: `21` (`WIN_SCORE`).
- Стартовая позиция мяча: центр поля (`BALL_INIT_X`, `BALL_INIT_Y`).

Эти параметры можно изменить в начале файла `pong21.c` и пересобрать проект.
